🎯 About KyaKre

KyaKre is an AI-powered career guidance platform built specifically for Indian students after 12th standard. Many students feel lost after 12th — not knowing which career to choose, which college to target, or what skills to build. KyaKre solves this by using Natural Language Processing and Machine Learning to match students with the most suitable career paths based on their interests, stream, and goals.

🤖 How the AI Works

Our recommendation engine uses TF-IDF (Term Frequency-Inverse Document Frequency) vectorization to convert career descriptions into mathematical vectors. It then calculates Cosine Similarity between careers to find the closest matches. The system was trained on 100+ Indian career profiles covering Science, Commerce, and Arts streams — with real salary data, top colleges, and skill requirements.

⚙️ Tech Stack

Python Flask Pandas Scikit-learn NLTK TF-IDF Cosine Similarity HTML/CSS/JS REST API

👨‍💻 Project Info

This project was developed as part of a full-stack internship focused on converting the static KyaKre website into a dynamic AI-powered career guidance platform. The backend handles NLP processing and API serving, while the frontend provides an intuitive multi-tab interface for students.
